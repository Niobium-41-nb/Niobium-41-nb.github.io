# Niobium-41-nb 个人网站

基于GitHub信息的个人网站，展示Niobium-41-nb的GitHub资料、项目和技能。

## 网站功能

- **响应式设计**：适配各种设备尺寸
- **GitHub信息展示**：显示用户资料、仓库统计等信息
- **项目展示**：展示GitHub上的项目
- **技能展示**：展示编程技能和专长
- **联系表单**：提供联系方式
- **平滑滚动**：增强用户体验

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 媒体查询)
- JavaScript (原生)
- Font Awesome 图标
- Google Fonts 字体

## 文件结构

```
.
├── index.html          # 主HTML文件
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── script.js      # JavaScript文件
├── README.md          # 项目说明
└── .nojekyll          # GitHub Pages配置
```

## 部署到GitHub Pages

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择分支（通常是main或master）
4. 网站将部署在：`https://niobium-41-nb.github.io`

## 本地开发

直接在浏览器中打开`index.html`文件即可查看网站。

## 自定义

要自定义此网站，请修改以下内容：

1. **个人信息**：在`index.html`中更新GitHub相关信息
2. **样式**：修改`css/style.css`中的颜色、字体等
3. **功能**：修改`js/script.js`中的JavaScript逻辑
